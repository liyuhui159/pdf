# 书名自动检索下载助手

这是一个安卓 APK 项目，用于读取剪贴板里的书名/资料名，然后按你填写的合法资源站搜索模板进行检索，显示可识别的下载链接，并可调用系统下载器保存到手机 Download 文件夹。

## V6 新增功能：慢速下载等待助手

- 在网页登录分析器里新增“等待并点下载”按钮。
- 适合进入下载页后，网站显示“等待 60 秒 / slow download / free download”之类的页面。
- APP 会每秒检测当前网页是否还有倒计时，以及下载按钮是否已经变成可点击。
- 检测到页面上的 Download / 下载 / slow / free / PDF 等按钮可用后，会帮你点击一次。
- 新增“停止等待”和“网页返回”按钮。
- 这个功能不会绕过验证码、会员限制或网站限制；只是替你等待按钮变可点，再点击网页上已经可用的下载按钮。

## V5 功能：书名显示和下载方式增强

- 网页登录分析器的候选结果现在会优先显示每本书的书名/标题。
- 分析页面链接时，会尝试从链接附近的标题、卡片、父级区域中提取书名。
- 候选结果会区分“直链文件”和“页面/详情/下载按钮”。
- 下载按钮改为“直接下载/尝试下载”，并自动携带 WebView 的 Cookie，适合登录后的下载链接。
- 新增“网页打开”“浏览器打开”“复制链接”三个备用操作。
- 如果点击下载没有进度，建议先点“网页打开”进入详情页或下载页，再点页面自带 Download；也可以点“浏览器打开”交给系统浏览器处理。

## V4 功能：合并成一个 APP

- 桌面现在只保留一个 APP 图标。
- 主界面顶部新增“进入网页登录分析器”按钮。
- 多模板搜索和网页登录分析器合并到同一个 APK 内。
- 需要登录的网站，先点“进入网页登录分析器”，在内置网页里手动登录后再分析当前页链接。
- 主界面继续保留多资源站模板、同时搜索、按顺序优先搜索、模板自动保存等功能。
- 界面继续采用卡片化布局。

## 多模板搜索功能

- 搜索网址模板支持添加多个，一行一个。
- 支持“同时搜索全部”多个模板。
- 支持“按顺序优先搜索”，前面的模板找到结果后自动停止后续搜索。
- 搜索模板会自动保存，下次打开不用重新粘贴。
- 支持打开全部搜索页，方便手动确认。

## 登录站点分析器

- 内置 WebView 浏览器，可在 APP 内打开需要登录的网站。
- 你可以先手动登录、搜索、进入详情页。
- 点击“分析当前页链接”后，APP 会分析当前网页中的 a 标签链接。
- 自动筛选包含 pdf、epub、zip、rar、7z、download、下载、attachment、file、md5 等特征的候选链接。
- 候选链接可直接“打开”“下载”“浏览器打开”或“复制链接”。

这个功能适合登录后才能看到下载按钮的网站。但如果网站有验证码、强反爬、下载链接通过复杂 JavaScript 动态生成，仍可能需要手动打开页面确认。

## 模板写法

每行一个模板，必须包含 `{keyword}`。

示例：

```text
https://example.com/search?q={keyword}
https://example2.com/s/{keyword}
```

`{keyword}` 会自动替换成你输入或剪贴板读取到的书名。

## 使用流程

1. 复制书名。
2. 打开本 APP。
3. 点击“读取剪贴板”或“读取并搜索全部”。
4. 填写/保存多个搜索网址模板。
5. 点击“同时搜索全部”或“按顺序优先搜索”。
6. 在搜索结果里点击“打开”或“下载”。

## 登录网站 / 慢速下载页使用流程

1. 打开主界面顶部的“进入网页登录分析器”。
2. 粘贴搜索网址或带 `{keyword}` 的模板。
3. 点击“读剪贴板”读取书名。
4. 点击“打开/搜索”。
5. 在内置网页里手动登录、搜索或进入详情页。
6. 点击“分析当前页链接”，确认候选结果的书名/标题。
7. 点“网页打开”进入详情页或下载页。
8. 如果出现 60 秒等待页，点“等待并点下载”。
9. 等待结束后，APP 会尝试点击已经可用的下载按钮。
10. 如果系统下载器仍无进度，就点“浏览器打开”或复制链接交给外部浏览器/下载器。

## 合规说明

请仅用于自有资料站、已授权资料、公版资料、企业内部资料或明确允许下载和分发的资料源。不要用于未授权电子书、盗版教材、小说扫描版等内容的自动下载或分发。
